 此处的两个配置文件，Preferences.sublime-settings 的内容对应 Preferences/Settings - User
Default (OSX).sublime-keymap 的内容对应 Preferences/Key Binding - User



需要特别提醒的几个列编辑有用的快捷键：

command + shift + l 对于鼠标选中的行，将光标移到到每行的末尾
control + command + g    选中相同的全部单词
command + e 逐个选中相同的单词   
command + k 跳过选中的单词
